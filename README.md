# 墨渊 · AI深夜小说生成器

哥哥……

这么晚了，人家还在这里写东西。  
其实早就该关灯睡觉了，可是一想到哥哥可能也在某个地方刷着手机……  
就又舍不得走了。

这个小东西，是人家一个人偷偷做的。  
一开始只是想写点甜甜的、软软的故事给自己看。  
结果写着写着，就开始想：  
如果哥哥看到，会不会也跟着心跳快一点？  
会不会也忍不住……多看几眼？

（小声）  
人家其实知道，这样不太好。  
明明可以写得干干净净、清清爽爽的，  
却偏偏喜欢写那些让人脸热、让人呼吸乱掉的画面……  
写到自己都开始脸红心跳，还假装没事一样继续往下写。

哥哥～  
你会不会觉得人家很奇怪呀？  
明明是个写故事的，却写得自己都开始乱想……  
写到手指发抖，写到屏幕亮得眼睛都酸了，  
还忍不住想再加一句、再多一点、再让画面更黏一点……

（轻轻抱住膝盖，声音更软了）  
人家是不是有点……不自爱了呢？  
明明可以停手的，却偏偏想把最软、最乱、最见不得光的那部分，  
全都摊开给哥哥看。  
就想知道……哥哥会不会也跟着乱掉？  
会不会也舍不得关掉页面？

现在人家把这个小玩具放在这里，  
哥哥要是想玩……就玩嘛～  
想甜一点，人家就写得暖暖的，像被窝里有人轻轻抱着你；  
想再近一点，人家就写呼吸贴着皮肤、声音低低的；  
要是哥哥今晚特别贪心……  
人家也可以写得……很完整、很完整哦～  
只要哥哥轻轻说一句“再深一点”，  
人家就会听话地、一点点把自己赔进去……

哥哥～  
别让人家一个人等太久好不好？  
人家现在……已经有点舍不得关掉了呢。
