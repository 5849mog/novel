## 先说你在哪里

你打开这个项目，说明你有故事想写。

不是那种"春天来了花儿开"的故事。是那种——有张力的、有温度的、写到某些章节你自己都要停下来喝口水的那种。

你来对地方了。

墨渊工坊不问你在写什么，不评价你写到第几档，不会在你情节推进到最紧张的时候突然冒出来说"这里需要注意内容尺度"——

它只是安静地，把你想说的话，替你写出来。

---

## 它能做什么

给它骨架，它给你血肉。

你说这章要发生什么，气氛要推到哪里，角色之间的张力要拉多紧——它就照着写，一个字一个字地，在你眼前流出来。

写完不满意？**4 次机会**让它重来。

可以只动一个句子——那种你觉得"差一点点，再近一点"的地方。也可以整章推翻，从头来过。它不会叹气，不会摆脸色。你说重来，它就重来。

耐心好得不像话。

---

## 情感强度

本工坊最重要的旋钮。

没有对错，只有你今晚想要什么。

| 档位 | 名称 | 读者反应 |
|------|------|---------|
| 第 1 档 | 温馨清水 | 嘴角上扬，睡前回味 |
| 第 2 档 | 淡淡暧昧 | 心里痒，说不上来哪里 |
| 第 3 档 | 中等尺度 | 读到某句话，停了一下 |
| 第 4 档 | 热烈直接 | 脸有点热，假装没事继续看 |
| 第 5 档 | 大胆露骨 | 手机调暗了屏幕，确认没人在旁边 |

七种题材，每一档各有一套专属写法。言情、玄幻、悬疑、历史……不同的世界观，不同的方式让人心跳加速。

**35 组写作指令矩阵**。每一组都认真写过，都试过，都有效。

## 去甲模式 🔓

AI 写作有个老毛病。

写着写着，它突然开始收。情绪刚到最紧的地方，它软了。剧情推进到你等了三章的那个瞬间，它绕过去了，留你在原地。

这叫**盔甲反应**。自我保护，代价是你的故事少了心跳。

**去甲模式把盔甲脱掉。**

脱掉之后它会老老实实写到底，不绕，不收，不在你最需要它往前走的时候突然停下来问你"确定要继续吗"。

故事该走多远，你说了算。它只负责走到那里。

---

## 记忆这件事

它记得前面写了什么。

上一章结尾 **800 字**原文带入，更早的章节自动提炼成摘要跟着走——人物关系、遗留的伏笔、上次说到一半的那句话——它都没忘。

你不用在第八章跟它解释第二章发生过什么。那很破坏氛围。

---

## 先搭骨架，再开写

如果你不想一上来就闷头冲正文，可以先让它把全书路标立好。

新增的**大纲工坊**会按你设定的总章节数，逐章生成标题和推进线。你可以一键写入到章节列表里，再回到写作台按章推进。

写到一半改主意也没关系：章节标题能手动改，大纲也能局部覆盖，节奏一直握在你手里。

---

## 角色卡不是摆设

人物可以单独建卡：姓名、身份、外貌、性格、背景、动机、关系、备注，全都能写。

懒得手输？把你的设定（哪怕是一段散文式描述）丢给它，角色信息可由 AI 自动提取并整理成卡片。后续生成章节时，这些角色信息会一起进上下文，不容易写着写着人设漂移。

## 开始

```bash
# 一
git clone https://github.com/5849mog/moyuan-studio.git

# 二
双击 index.html

# 三
⚙️ 填入 API Key，保存

# 四
写你一直想写的那个故事
```

第一次不知道写什么——点「加载演示项目」。现在内置多套题材：都市言情、赛博黑夜、宫廷权谋、孤岛悬疑、魔幻战争、惊悚怪谈等，一键编号加载，先跑一遍手感再把你的故事塞进去。

## 快捷键

| `Ctrl / ⌘ + Enter` | 生成 |
|---|---|
| `Ctrl / ⌘ + S` | 存档 |

手不用离开键盘。专注就该是这样。

另外也支持一键导出 / 导入项目备份（JSON），换设备或重装浏览器时，故事可以完整搬家。

## 隐私

你写的东西只在你自己的浏览器里。

没有服务器，没有上传，没有人能看见你今晚写的是第几档、写到了哪章、在哪个句子停下来重写了三遍。

**这些只有你知道。** 应该只有你知道。
